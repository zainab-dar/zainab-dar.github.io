Project Overview: UNCC Muslim Technology Collaborative

Project Overview

The UNCC Muslim Technology Collaborative (MTC)is a student-led organization that brings together Muslim students with an interest in technology. The club hosts speaker panels, technical skill-building workshops, social events, and discussions that explore how faith, community, and innovation intersect. The goal of this web application is to create an interactive platform that improves communication, promotes engagement, and provides clear access to event details, leadership information, and opportunities to get involved.

Purpose

This website will serve as a hub for current members, new students, and potential collaborators to learn about the club’s initiatives, view upcoming events, and connect with others interested in technology. It will help increase visibility for the organization and streamline communication through structured event information and a membership form.

Intended Users

  • Current UNCC students interested in technology and community building.
  • Muslim students seeking a supportive and inclusive space to explore tech topics.
  • Potential guest speakers, sponsors, or partners who want to collaborate with the organization.

Overview of Website Content

  • Home: Overview of the club, featured images, and highlights of upcoming events.
  • About Us: Information about the club’s mission, leadership, and history.
  • Events: Calendar and details for upcoming and past events.
  • Resources: Technical learning materials, speaker recordings, and links to helpful websites.
  • Join Us / Contact: Membership interest form and contact information for the executive board.

Page Design Details

Home Page

  • Purpose: Introduce MTC and display highlights of events and updates.
  • Users: General visitors and current members.
  • Content: Club overview, event highlights, welcome message, and navigation links.
  • Data Entry: None.
  • Interactive Elements: jQuery image carousel and animated “Join Us” button.

About Us

  • Purpose: Explain the club’s mission, history, and leadership team.
  • Users: Visitors wanting to learn more about the organization.
  • Content: Text sections and officer photos with bios.
  • Interactive Elements: Accordion or collapsible panels for officer bios using jQuery UI.

Events

  • Purpose: Display information about upcoming and past events.
  • Users: Members and students interested in attending events.
  • Content: Event name, date, description, and category (e.g., panel, workshop, social).
  • Data Entry: Optional search/filter bar.
  • Interactive Elements: AJAX event filtering by category.

Resources

  • Purpose: Provide members with learning resources and recordings from past events.
  • Users: Club members and students interested in self-learning.
  • Content: Links to resources, tutorials, and embedded media.
  • Interactive Elements: Hover tooltips and collapsible resource sections.

Join Us / Contact

  • Purpose: Collect interest and contact information from potential members.
  • Users: Prospective members, sponsors, and collaborators.
  • Content: Input fields for name, email, and interest area.
  • Validations: Required fields validation using JavaScript.
  • Interactive Elements: Form validation and confirmation popup after submission.

Dynamic Functionality

This project includes several interactive features using JavaScript, jQuery, and AJAX to improve user engagement and functionality:

  • Home Page: jQuery image carousel for event highlights.
  • About Us: jQuery UI accordion for officer bios.
  • Events Page: AJAX event filtering by category.
  • Join Us Page: JavaScript form validation with alert confirmation.
  • Resources Page: Interactive gallery using hover effects and expandable panels.
