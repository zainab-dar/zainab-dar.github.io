FCC Full Stack JavaScript Outline

Section Title Steps Description
1. Code Editors5 stepsIn these lessons, you will learn how to work with code editors and IDEs. You will learn various concepts about the most popular code editor, VS Code such as its installation, how to create a project in it, keyboard shortcuts, and extensions.
2. Variables and Strings62 stepsIn these lessons, you will learn the fundamentals of JavaScript. Topics covered include, but are not limited to, variables, data types, how JavaScript interacts with HTML and CSS, and much more.
3. Booleans and Numbers37 stepsIn these lessons you will learn about the number type, arithmetic operators, and using them with numbers and strings.
4. Functions38 stepsIn these lessons, you will learn how to reuse a block of code with functions. You will learn what the purpose of a function is and how they work, and how scope works in programming.
3. Arrays35 stepsIn these lessons, you will learn how to work with JavaScript arrays. You will learn about what makes an array, one-dimensional and two-dimensional arrays, how to access and update the elements in an array, and much more.
5. Objects28 stepsIn these lessons, you will learn the fundamentals of JavaScript objects, including how to create them, access their properties, and understand the difference between primitive and non-primitive data types.
6. Loops22 stepsLoops are an essential part of JavaScript. That's why the following lessons have been prepared for you to learn about the different types of loops and how they work, and also how iteration works.
8. JavaScript Fundamentals Review30 stepsIn these lessons you will learn about string objects, the toString() method, the Number constructor and more.
7. Higher Order Functions and Callbacks37 stepsIn these lessons, you will learn how to work with higher order functions and callbacks. The higher order functions you will learn include map(), filter(), reduce(), sort(), every(), and some(). You will also learn how to chain these methods together to achieve your desired results.
8. DOM Manipulation and Events87 stepsIn these lessons, you will learn how to work with the Document Object Model (DOM), the addEventListener() method and events, and web APIs.
9. JavaScript and Accessibility42 stepsIn these lessons you will learn more about ARIA attributes like aria-expanded, aria-live, and common ARIA states.
10. Debugging8 stepsIn these lessons, you will learn about the common errors in JavaScript and the techniques you can use to fix them – a process called debugging.
11. Basic Regex47 stepsIn these lessons, you will learn about regular expressions in JavaScript. You will learn about the methods for working with regular expressions, modifiers, character classes, lookaheads, lookbehinds, back-references, quantifiers, and more.
12. Form Validation103 stepsIn these lessons, you will learn about form validation in JavaScript. You will learn about the various ways to validate forms, how the preventDefault() method works, and how the submit event works.
13. Dates5 stepsIn these lessons, you will learn about the JavaScript date object. You will learn about the methods for working with dates and how to format dates.
14. Audio and Video Events55 stepsIn these lessons, you will learn how to work with audio and video files using JavaScript. You will learn about the Audio and Video constructors, their methods and properties, audio and video formats, codecs, the HTMLMediaElement API, and much more.
15. Maps and Sets37 stepsIn these lessons, you will learn about JavaScript Map and Set. You will also learn how they both differ from WeakSets and WeakMaps.
16. localStorage and CRUD Operations78 stepsIn these lessons, you will learn about client-side storage and CRUD operations in JavaScript. You will learn about localStorage and sessionStorage alongside their methods and properties, cookies, the Cache API, IndexDB, and much more.
17. Classes68 stepsIn these lessons, you will learn about classes in JavaScript. You will learn about inheritance, the this keyword, static properties and methods, and more.
18. Recursion112 stepsIn this lesson, you will learn about recursion and the call stack.
19. Functional Programming23 stepsIn these lessons, you will learn about functional programming and how to nest functions using a technique called currying.
20. Asychronous JavaScript36 stepsIn these lessons, you will learn about asynchronous programming in JavaScript. You will learn about the differences between synchronous and asynchronous programming, how the async keyword works, the Fetch API, promises, async/await, the Geolocation API, and much more.
